# Cat Generator using DCGAN
The model was trained on 200 epochs with small batches. 

## Training
To run the training
```sh
python ./src/train_dcgan.py
```
## Generate fake samples
To generate 64 cats from dcgan
```sh
python ./src/generate_fake.py
```
